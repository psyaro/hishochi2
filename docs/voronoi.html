星空の明るさと光害（ボートル・スケール）

本システムで表示している「星空の明るさ」は、夜空の暗さを示す物理指標である SQM (Sky Quality Meter) 値（単位: mag/arcsec²）および、アマチュア天文学で広く使われる夜空の暗さの指標である ボートル・スケール (Bortle scale) に基づいています。

ボートル・スケールと天の川の見えやすさ

数値が大きいほど人工光による光害が深刻で、星が見えにくくなります。

クラス 夜空の分類 (Label) SQM基準 天の川の見えやすさ
Class 1 卓越して暗い空 (Excellent dark sky) 21.99–22.00 天の川が肉眼で極めて鮮明に見え、三角座銀河(M33)も容易に観察できます。最高峰の夜空です。
Class 2 平均的な暗い空 (Average dark sky) 21.89–21.99 天の川が非常に鮮明で、中心部の複雑な構造や「ちり星雲」の微細な明暗が肉眼で確認できます。
Class 3 田舎の空 (Rural sky) 21.69–21.89 天の川の構造がよく見えます。地平線付近にわずかに光害を感じる程度で、天頂は十分に暗いです。
Class 4 田舎と郊外の境界 (Transition) 20.49–21.69 天の川は天頂付近で高いコントラストを持ち、複雑な構造が確認できます。地平線近くには光害が見えます。
Class 5 郊外の空 (Suburban sky) 19.50–20.49 天の川は天頂付近で非常に弱く、白っぽくかすんで見える程度です。地平線は光害ドームに囲まれます。
Class 6 明るい郊外の空 (Bright suburban) 18.94–19.50 天の川は天頂付近でのみかろうじて見える程度です。空全体が光害で白く明るいです。
Class 7 郊外と都市の境界 (Transition) 18.38–18.94 天の川は天頂であっても完全に見えません。空全体が光害で覆われ、淡い天体は見えません。
Class 8-9 都市・都心の空 (City / Inner city) 18.38未満 光害が極めて深刻で、天の川は完全に見えません。主要な明るい一等星のみが確認できます。
